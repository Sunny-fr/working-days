# CLI Tool : Working days

![GitHub Logo](/_assets/screenshot.jpg)

## requirements
Requires Node.JS 8+

## Clone
```
git clone https://github.com/Sunny-fr/working-days
```

## First time only
```
$ npm install
```

## Running
```
$ npm start
```
or

```
$ node cli.js
```

## Changing the month
(ex for 2019 may)

```
$ node cli.js 5 2019
```

## Run it anywhere
typing this turns it into cli classic command
```
$ npm link
```
then run it from anywhere
```
$ jour-ouvre
```
